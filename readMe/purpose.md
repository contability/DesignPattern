	· 기능의 사용 통로
#####· 델리게이트 개념을 이해한다
	· 떠넘기다, 위임하다라는 뜻
	· 인터페이스의 기능을 수행하게 되면 수행 시킨 애는 정확히 어떤 동작을 하는지 모르지만 
	이 인터페이스를 상속 받아 오버라이딩 한 인스턴스의 기능을 가져와서 알아서 사용하게 될 것
	이 때 해당 인터페이스는 기능을 사용할 수 있게 하는 접근점이 된다.
#####· 전략(strategy) 패턴 개념을 이해한다
